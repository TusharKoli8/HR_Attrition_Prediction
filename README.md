
**HR Attrition Prediction** 

 **Overview**

A machine learning project to predict employee attrition and identify key factors affecting retention.

**Objective**

Predict whether an employee will leave
Identify important drivers of attrition

**Tech Stack**

Python, Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn

**Models**
Logistic Regression
Decision Tree
Random Forest (Best Model)

**Results**

Random Forest achieved highest performance
Key factors: Overtime, Salary, Distance, Job Satisfaction

** Conclusion**

Helps HR teams make data-driven decisions to reduce employee attrition.
